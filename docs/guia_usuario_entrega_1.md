# Guia de usuario - Entrega 1

## Ingreso

1. Abrir FEMAG Desktop.
2. Ingresar usuario y clave.
3. El menu visible depende del perfil asignado.

## Clientes y domicilios

1. Entrar en Maestros > Clientes.
2. Cargar razon social o nombre, CUIT, condicion IVA, telefono, email y contacto.
3. Agregar un domicilio fiscal.
4. Agregar uno o mas domicilios de entrega.
5. Marcar solo un domicilio de entrega como principal.

## Maestros operativos

Desde Maestros se cargan productos, choferes, transportistas, camiones, tipos de pallet y servicios.
Las altas y cambios quedan auditados.

## Accesos futuros

Ordenes de carga, remitos, F150, pagos, recibos y cuenta corriente quedan visibles como accesos previstos para una proxima entrega.
Al usarlos se informa: `Funcionalidad prevista para una próxima entrega.`

## Auditoria

El administrador puede revisar acciones registradas: login, usuarios, permisos, clientes, domicilios, maestros y backups manuales.

## Backups

El administrador puede ejecutar backup manual. El sistema registra fecha, estado y archivo generado.
